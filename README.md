# piggy-bank


## Setup

1. Install all dependencies 
 ```npm i```

2. Run the command:
 ```npm run dev```

3. Check for errors:
 ```npm run lint```

 ### Endpoints
  
  - ``GET`` - ``/user`` get all users;
  - ``GET`` - ``/user/id`` get user by id;
  - ``POST`` - ``/user`` create new user;
  - ``PUT`` - ``/user/id`` update user;
  - ``DELETE`` - ``/user/id`` delete user;
